## 博客简介

Galenry 聚焦三类内容：

- 办公效率与实战技巧：把复杂流程拆成新手也能上手的步骤。
- 电脑硬件/软件安装与调优：重视真实场景、参数对比和避坑建议。
- AI 资讯与工具落地：不只讲概念，更关注普通人怎么直接用起来。

如果你希望用更少时间，获得更高质量的信息密度，这里的文章会很适合你。

## 定期更新机制

每次主站发布新文章时，在上面的目录里追加一行即可。建议格式：

## GitHub Pages 镜像摘要

本仓库提供了一个简单的 [index.html](./index.html) 作为内容摘要页，可用于开启 GitHub Pages。页面中已设置 canonical：
